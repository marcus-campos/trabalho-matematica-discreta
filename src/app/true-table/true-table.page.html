Existem gatos verdes. Pode-se observar que: É uma proposição...é uma frase verdadeira. Não é uma proposição, pois é apenas uma pergunta e não pode ser classificada como falsa e nem verdadeira. É uma proposição...embora seja necessário uma pesquisa para confirmar se é falsa ou verdadeira. 2) Conectivos Conectivos, de modo geral, são formas de ligações entre proposições. A B ( A implica B ) significa também: B decorre de A; A é condição suficiente para B; Se A , então B; Na implicação é um pouco mais difícil de se verificar quando é verdadeiro ou falso, portanto, considere o seguinte : a implicação só será falsa se o antecedente for verdadeiro e o conseqüente falso. Supondo A: passar no vestibular e B: ganhar um carro, a implicação A B significa que ganhar o carro decorre do fato de ter passado no vestibular. Se não passar no vestibular pode-se ganhar o carro ou não, por isto é convencionado ser verdadeiro. Só é falso se passar no vestibular e não ganhar o carro. Notas: 1) Condição suficiente. P é condição suficiente para Q significa: se P, então Q ( P Q ) Note ainda que numa sentença, em português, o antecedente pode vir depois do consequente. Assim, em “ uma condição suficiente para interromper a energia elétrica é desligar a chave geral”, P é desligar a chave geral( Antecedente) e Q é interromper a energia elétrica( Consequente). Se desligar a chave geral, então interrompe a energia elétrica (P Q). 2) Condição necessária: P é uma condição necessária para Q significa: se não P, então não Q ~P ~Q ou, como será visto adiante, Q P Na sentença “ fogo é condição necessária para fumaça” tem-se: Se não há fogo, logo não há fumaça. ~ P ~ Q Se há fumaça, então há fogo. 3) A equivalência sempre retrata uma condição necessária e sufiente.
